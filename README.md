## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```
visit: http://localhost:5173/

### 3. Build for production

```bash 
npm run build
```

### 4. Linting

```bash
npm run lint
```
